{% trans "Quick Start" %}

{% blocktrans %}Install and make predictions in 5 minutes{% endblocktrans %}

  • PIP (Python)
  • Docker

{% blocktrans %}Tested on CentOS 7, Ubuntu 16.04, and MacOS 10.12, with Python 2.7.x installed{% endblocktrans %}

{% blocktrans %}Download {% endblocktrans %}{% blocktrans %}the trained housing prices model{% endblocktrans %}{% blocktrans %} and then install PaddlePaddle on your computer:{% endblocktrans %}

pip install paddlepaddle

{% blocktrans %}Download, and then install Docker on your computer.{% endblocktrans %} {% blocktrans %}After installing Docker, create a new file called housing.py in a new and empty directory:{% endblocktrans %}

mkdir ~/workspace
cd ~/workspace; touch housing.py

{% blocktrans %}Instructions on installing pip{% endblocktrans %}

{% blocktrans %}Now, create a new file called {% endblocktrans %}housing.py{% blocktrans %}, and paste this Python code{% endblocktrans %} {% blocktrans %}(make sure to set the right path based on the location of fit_a_line.tar on your computer):{% endblocktrans %}

{% blocktrans %}Now download {% endblocktrans %}{% blocktrans %}the trained housing prices model into the ~/workspace directory, and paste this Python code in housing.py using your favorite editor{% endblocktrans %}:

import paddle.v2 as paddle

# Initialize PaddlePaddle.
paddle.init(use_gpu=False, trainer_count=1)

# Configure the neural network.
x = paddle.layer.data(name='x', type=paddle.data_type.dense_vector(13))
y_predict = paddle.layer.fc(input=x, size=1, act=paddle.activation.Linear())

with open('fit_a_line.tar', 'r') as f:with open('/workspace/fit_a_line.tar', 'r') as f:
    parameters = paddle.parameters.Parameters.from_tar(f)

# Infer using provided test data.
probs = paddle.infer(
     output_layer=y_predict, parameters=parameters,
     input=[item for item in paddle.dataset.uci_housing.test()()])

for i in xrange(len(probs)):
     print 'Predicted price: ${:,.2f}'.format(probs[i][0] * 1000)

{% blocktrans %}Run {% endblocktrans %}python housing.py{% blocktrans %} and voila! It should print out a list of predictions for the test housing data.{% endblocktrans %}

{% blocktrans %}Now run this code in a new PaddlePaddle Docker container:{% endblocktrans %}

docker run --rm -v ~/workspace:/workspace paddlepaddle/paddle:latest python /workspace/housing.py

{% blocktrans %}Voila! It should print out a list of predictions for the test housing data.{% endblocktrans %}
